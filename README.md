![Screenshot]([https://placehold.co/720x400/111118/00ffcc?text=Neon+Portfolio+Preview](https://avatars.mds.yandex.net/i?id=3b1f228913e081cfba067d2b090edda2_l-10936100-images-thumbs&n=13))

## ✨ Features

- **Neon UI/UX** – Glowing gradients, dark theme, and hover animations
- **Responsive Layout** – Perfectly adapts to desktops, tablets, and phones
- **Project Grid** – Square cards with icons, titles, and descriptions
- **Interactive Elements** – Smooth hover effects and tactile click feedback
- **Customizable** – Easy to update your name, bio, and project links
- **Gojo Satoru** - HAS BEEN SEALED

## 🚀 Quick Start

1. **Clone the repository**
   ```bash
   git clone https://github.com/yourusername/neon-portfolio.git
   cd neon-portfolio
Open the HTML file

Simply double-click index.html or open it with your favorite browser.

Customize content

Edit the profile section (name, bio, badge)

Update project cards with your own links and descriptions

Replace placeholder icons and text

🎨 Customization Guide
Profile Information
html
<div class="greeting">
  FED4IK                          <!-- Your name -->
  <small>My projects</small>      <!-- Subtitle -->
</div>
<div class="badge">Developer</div> <!-- Badge text -->
<div class="profile-description">
  🚀 <span>Full-stack developer</span> · your bio here
</div>
Project Cards
Each project card follows this structure:

html
<div class="project-item">
  <a href="https://suno.com/s/hmLzKrpXl8LqDmoh" class="project-link" target="_blank">
    <span class="icon">🎵</span>        <!-- Emoji or SVG icon -->
    <span class="info">
      <span class="name">Project Name</span>
      <span class="desc">Short description</span>
    </span>
    <span class="arrow">→</span>
  </a>
</div>
Colors & Theme
The neon color palette is defined in CSS variables and gradients:

Primary: #00ffcc (cyan/teal)

Secondary: #8800ff (purple)

Background: #0a0a0f (dark)

Text: #e0e0ff (light blue)

📱 Responsive Breakpoints
Device	Width	Layout
Desktop	> 600px	2-column grid
Tablet	600px	1-column grid, larger cards
Mobile	< 400px	1-column grid, compact cards
